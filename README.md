# Portfolio Robert Griffaton - BUT3 S6

Ce projet est le portfolio professionnel et académique de Robert Griffaton pour la validation de son BUT3 Informatique (Semestre 6) et son admission à l'ESIEE Paris.

## 🚀 Stack Technique
- **Framework :** React 19 + Vite
- **Langage :** TypeScript (Typage strict)
- **Styles :** Tailwind CSS 4 (Architecture CSS-first)
- **Animations :** Framer Motion
- **Icons :** Lucide React

## 🏗️ Architecture (Clean Code)
Le projet suit les principes SOLID et DRY avec une structure découplée :
- `src/components/` : Composants UI atomiques.
- `src/sections/` : Sections de la One-Page.
- `src/layouts/` : Layout principal (Navigation, Footer).
- `src/services/` : Abstraction des données via le **Repository Pattern**.
- `src/types/` : Définitions TypeScript globales.
- `src/data/` : Données statiques (Mock Data).

## 📂 Documentation & Mémoire
- `CONTEXT.md` : Journal de bord et mémoire du projet (à consulter par l'IA et l'utilisateur).
- `DOC/` : Dossier contenant les traces d'apprentissage et auto-évaluations.

## 🛠️ Installation
```bash
npm install
npm run dev
```
